Passo a Passo para Montagem do Projeto

1. Reunir os Materiais

Certifique-se de ter todos os componentes necessários para o projeto.

Lista de Materiais:

  • 1 Placa Arduino (UNO)
  • 1 Sensor Ultrassônico
  • 1 Protoboard
  • Cabos Jumper
  • 1 Resistor de 330Ω
  • Fonte de alimentação USB para o Arduino
  • Computador com IDE Arduino instalada

2. Conexões do Sensor Ultrassônico

  • VCC --- 5V
  • GND --- GND
  • TRIG --- Pino Digital 5
  • ECHO --- Pino Digital 4

2. Conectar a Protoboard ao Arduino

Ligue os terminais de alimentação da protoboard do Arduino. Verifique se todas as conexões estão firmes e nos pinos corretos.

3. Montagem do Sensor Ultrassônico

Fixe o sensor ultrassônico na protoboard e conecte os cabos conforme o esquema do projeto.

4. Conexão com o Computador

Conecte o Arduino ao computador via cabo USB e abra o software de monitoramento.

5. Configurar a IDE Arduino e Carregar o Código

  • Instale a [IDE Arduino](https://www.arduino.cc/en/software).
  • Configure a porta COM correspondente ao seu Arduino.
  • Selecione a placa correta (UNO) no menu "Ferramentas".
  • Abra a IDE Arduino no computador, copie o código abaixo e cole na IDE.
  • Clique em "Upload" para carregar o código na placa Arduino.

Código em C++ para o Arduino

        
const int trigPin = 5; // Emite o sinal sonoro
const int echoPin = 4; // Capta as ondas sonoras refletidas
int enviarpy;

void setup() {
  Serial.begin(9600);
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
}

void loop() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(5);
  digitalWrite(trigPin, LOW);

  long duracao = pulseIn(echoPin, HIGH);
  float distancia = duracao * 0.034 / 2;

  if (distancia > 80) {
    enviarpy = 1;
    Serial.print(enviarpy);
  } else {
    enviarpy = 0;
    Serial.print(enviarpy);
  }

  Serial.print("Distância: ");
  Serial.print(distancia);
  Serial.println("cm.");

  delay(500);
}

Código em Python para Integração

  • Abra a IDE VS Code no computador, copie o código abaixo e cole na IDE.
  • Salve o código Python em um arquivo ("bloqueio.py").
  • Execute o script no terminal:
    • "bash python bloqueio.py"
  • Teste o sistema movendo-se próximos ao sensor e observe o bloqueio de tela.
        
import serial
import os
import time

# Configuração da porta serial
arduino = serial.Serial('COM3', 9600)  # Substitua 'COM3' pela porta correta

def bloquear_tela():
    print("Bloqueando a tela...")
    if os.name == 'nt':
        os.system("rundll32.exe user32.dll,LockWorkStation")
    else:
        os.system("gnome-screensaver-command -l")

try:
    while True:
        if arduino.in_waiting > 0:
            dados = arduino.readline().decode('utf-8').strip()
            print(f"Dados recebidos: {dados}")
            if dados == '1':
                bloquear_tela()
            time.sleep(1)
except KeyboardInterrupt:
    print("Encerrando...")
    arduino.close()
