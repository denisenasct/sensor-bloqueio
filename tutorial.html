Aplicativos e Plataformas Utilizados

Arduino IDE

Software para programação e upload do código no Arduino.

Python

Script para integração com o sistema operacional do computador. Versão 3.12.

GitHub

Repositório para hospedagem e compartilhamento do código.

Protoboard

Plataforma para montagem do circuito eletrônico.

Paint

Ferramenta para ajustes e criação de esquemas simples.

VS Code

Editor de código usado para o desenvolvimento do projeto. (Versão: 1.95.3)

YouTube

Plataforma para vídeos explicativos e tutoriais.

JavaScript

Linguagem utilizada para adicionar e melhorar a experiência visual do site.
