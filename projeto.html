Projeto Bloqueio de Tela Windows

Baixar Projeto

Descrição do Projeto

Motivação

Em muitas empresas, existe uma política de segurança que exige que os profissionais bloqueiem suas telas ao se afastarem de seus computadores. Isso evita que terceiros acessem informações privadas e utilizem indevidamente contas que não lhes pertencem.

Objetivo

O objetivo deste projeto é desenvolver um sistema que detecte automaticamente quando o usuário se afasta do computador e, ao alcançar uma distância definida pelo sistema, bloqueie a tela usando o comando Win + L. Isso será feito com o auxílio de um sensor ultrassônico e uma placa Arduino, promovendo segurança e facilitando o cumprimento das políticas corporativas.

Contextualização

Este projeto foi desenvolvido pelos alunos do primeiro período de Sistemas de Informação, na disciplina de Introdução à Computação e Sistemas Digitais. Utilizamos uma placa Arduino, um sensor ultrassônico e uma protoboard para criar um dispositivo que monitora a presença do usuário. Este sistema é ideal para aplicações em ambientes empresariais, onde a segurança de informações e contas de usuário é essencial.

Esquema do Projeto

Descrição do Esquema

Sensor Ultrassônico:

  • Emite ondas ultrassônicas para medir a distância do usuário.
  • Retorna os dados de distância ao Arduino.

Arduino:

  • Processa os dados recebidos do sensor.
  • Envia comandos ao computador via USB.

Computador:

  • Executa o script Python para monitorar os comandos do Arduino.
  • Bloqueia a tela se a distância atingir o limite especificado.
